# Fisheriescape pivot table 3 in path to CEU calculations.

Fisheriescape pivot table 3 in path to CEU calculations.

## Usage

``` r
fs_pivot3(pivot2, group.cols, id.col = NULL, trap.fishery)
```

## Arguments

- pivot2:

  The output from function `fs_pivot2`

- group.cols:

  Names of columns to group by.

- id.col:

  Name of column that is the vessel or licence or individual specifier.

- trap.fishery:

  Is it a trap fishery? Default is 'yes'.
