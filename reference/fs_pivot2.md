# Fisheriescape pivot table 2 in path to CEU calculations.

Fisheriescape pivot table 2 in path to CEU calculations.

## Usage

``` r
fs_pivot2(pivot1, trap.fishery = "yes", group.cols)
```

## Arguments

- pivot1:

  The output from function `fs_pivot1`

- trap.fishery:

  Is it a trap fishery? Default is 'yes'.

- group.cols:

  Names of columns to group by.
